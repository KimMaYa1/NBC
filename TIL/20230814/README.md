## 기초 사용 문법

 - Console.WriteLine();  
   - 콘솔창에 ()칸에 해당하는 텍스트를 쓰고 다음줄로 넘어가!
 - Console.Write();      
  -  WriteLine과 차이점은 다음줄로 넘어가지 않는것!
 - Console.ReadLine();   
  - 콘솔창에 플레이어가 입력할수 있게 해줘!
 - Console.Clear();
  - 콘솔창에 사용된 텍스트를 모두 지워!
